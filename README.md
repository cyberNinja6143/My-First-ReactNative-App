# My first react native app

As the title suggests this application uses React Native. The purpose of this app is to practice connecting mobile applications to backend.

This app demonstrates the following.

- User login and account creation.
- Email verification.
- Simple display of user account page.
- Uploading of photos to database.
- Viewing of uploaded photos.
- Ability to delete account.

Feel free to use this for future reference and if you are not future me then feel free to use this however you like.
